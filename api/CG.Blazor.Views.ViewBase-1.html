    Remarks

    The idea, with this class, is to create a simple MVVM integration for use with Blazor. As we all know, Blazor was deliberately designed not favor any one paradigm over another, so, if we want to use MVVM with Blazor, which we do, then we're on our own for that part. That's fine though because, the only parts that are really missing are property change notifications, and we'll handle those here.

    Examples

    All that's required to use this class is to add the following tag to your razor page markup:

    @inherits ViewBase{IMyViewModel}

    Where IMyViewModel is, whatever your view-model type is.
